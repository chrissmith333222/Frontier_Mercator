# Parallax Dashboard — Frontier Mercator Group

Real-time geopolitical intelligence visualization for Africa and Latin America.

## Running Locally

### Prerequisites
- Python 3.9+
- pip

### Installation

1. Navigate to the project directory:
```bash
cd "C:\Users\chris\OneDrive\Desktop\Frontier Mercator"
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the dashboard:
```bash
streamlit run dashboard.py
```

The dashboard will open in your browser at `http://localhost:8501`.

## Features

- **🗺️ Severity Map** — Geographic visualization of all events across Africa and Latin America
- **📊 Analytics** — Severity distribution, regional breakdown, time series
- **🔥 Critical Events** — High-severity events (≥7) requiring immediate attention
- **ℹ️ About** — Methodology and data source documentation

## Data

The dashboard reads from `data/normalized/acled_latest_normalized.json`, which is generated by the ACLED ingestion pipeline:

```bash
python scripts/ingestion/acled_fetch.py --days-back 3000 --output data/raw/acled_latest.json
python scripts/ingestion/acled_normalize.py --input data/raw/acled_latest.json --output data/normalized/acled_latest_normalized.json
```

## Deployment

### Streamlit Community Cloud (Free)

1. Create a GitHub account and push this repository to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click "New app" and select your repository
4. Streamlit will deploy automatically

### Custom Domain

Once deployed on Streamlit, you can point your custom domain (`frontiermercator.com`) to the Streamlit URL via CNAME record.
